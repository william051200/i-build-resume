# Features

## Tech stack

- **Vue 3** (`<script setup>` single-file components) for the reactive UI
- **Vite** for the dev server and production build
- **TypeScript** for a type-safe resume model
- No backend — everything runs locally in the browser; data is stored in `localStorage`

## Features

- **Top-to-bottom layout**: Details → Summary → Experience → Education → Skills → Projects → Certifications
- **Live preview** updates as you type (reactive, no manual DOM wiring)
- **Auto-save** to your browser (localStorage) — reload and keep editing
- **Save / Print PDF** using your browser's built-in print dialog (clean, single-column, ATS-friendly)
- **Load sample** data to see how it looks
- **Import / Export JSON** — load a resume from a `.json` file (auto-fills every section) or download your current resume as a JSON file
- **Clear saved data** button to start fresh
- No profile picture, no clutter — just a polished, professional resume

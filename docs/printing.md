# Export to PDF

1. Click **Save / Print PDF** in the top bar (or press `Ctrl + P`).
2. In the print dialog, set **Destination / Printer** to **Save as PDF**.
3. Recommended settings:
   - Paper size: **A4**
   - Margins: **Default**
   - **Uncheck** "Headers and footers"
   - **Check** "Background graphics" (so the accent colors and skill tags print)
4. Click **Save** and choose where to store your PDF.

Only the resume itself prints — the editor form and buttons are hidden automatically.
